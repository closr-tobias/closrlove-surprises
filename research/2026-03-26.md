## 5 V2 Block Ideas

### 1. 🌌 Constellation Draw
**Interaction:** User taps/drags on a dark sky to place stars. Lines auto-connect nearby stars into constellations. After ~8-12 stars, the constellation "names itself" with the hidden message, glowing in as a label. Stars twinkle with staggered CSS animation.
**Emotional purpose:** "You mapped our own sky." Co-creation, intimacy, making something unique together.
**Technical approach:**
- Canvas 2D for star placement + nearest-neighbor line connections within distance threshold
- CSS `mix-blend-mode: screen` for glow; `@keyframes` twinkle with randomized `animation-delay`
- Message reveal: `opacity` transition on text positioned at constellation centroid
- Optional parallax drift on `deviceorientation` (graceful fallback to mouse)

---

### 2. 🫧 Breath Sync
**Interaction:** A soft glowing orb pulses. "Breathe with me." User holds finger on screen — orb expands (inhale). Release — orb contracts (exhale). After 3-4 cycles, orb dissolves into particles that reform as the hidden message.
**Emotional purpose:** Forced intimacy through shared rhythm. Calming. The message feels *earned* through presence.
**Technical approach:**
- `touchstart`/`touchend` to track hold duration
- Orb: radial gradient div with `transform: scale()` driven by rAF easing
- Box-shadow layering for soft glow (3-4 shadows, increasing blur)
- Particle dissolution: ~200 tiny divs, FLIP to text character positions
- State machine: `idle → inhale → exhale` × 3 → `reveal`
- Optional: `navigator.vibrate(50)` on each exhale

---

### 3. 🎵 Sound Garden
**Interaction:** Dark space with faint grid. User taps to plant "sound flowers" — each spawns a procedural CSS flower emitting a musical tone. Different X positions = different pentatonic notes. After 6+ flowers, "bloom" button appears — all burst open revealing message petals.
**Emotional purpose:** Playful creation. Synesthesia of sight + sound.
**Technical approach:**
- Web Audio API: `OscillatorNode` pentatonic mapping by tap X; `GainNode` envelope for pluck
- Flowers: CSS `clip-path` polygons, random rotation/hue via custom properties
- Bloom: `transition: transform 0.6s` scale-up + `::after` pseudo-elements with message chars

---

### 4. 🧊 Ice Melt
**Interaction:** Message frozen behind translucent ice (frosted glass). User rubs finger across surface to "melt" it — frost clears where they touch. Tiny water droplets slide down with gravity.
**Emotional purpose:** Anticipation through effort. Slow reveal builds tension. Feels physical and satisfying.
**Technical approach:**
- Frost: `backdrop-filter: blur(20px) brightness(1.2)` + SVG `feTurbulence` noise texture
- Canvas overlay: `globalCompositeOperation: 'destination-out'` erases frost on touch path
- Droplets: small divs animated downward with CSS transform + slight X wobble
- At 60%+ cleared → frost auto-dissolves
- Limit ~30 active droplets, recycle oldest

---

### 5. 🎐 Wind Chime
**Interaction:** Dangling chime elements respond to device tilt (gyroscope) or mouse movement. Tilting phone makes chimes swing and clink with soft tones. Each collision reveals one word of the message on the chime surface.
**Emotional purpose:** Ambient discovery. Message is *found* through gentle play. Meditative, surprising.
**Technical approach:**
- `DeviceOrientationEvent` → map `gamma` to pendulum force
- Simple pendulum physics: angle, velocity, gravity, damping per frame
- Each chime: `transform-origin: top center`, `transform: rotate(${angle}deg)`
- Collision: compare adjacent chime angles; trigger Web Audio sine pluck on contact
- Word reveal: `opacity: 1` transition on collision
- Fallback (no gyro): auto-sway + tap to nudge

## Priority Ranking
1. **Ice Melt** — strongest tactile feel, universally works, clear mechanic
2. **Breath Sync** — unique emotional hook, simple to build, powerful intimacy
3. **Constellation Draw** — creative co-creation, gorgeous visually
4. **Wind Chime** — beautiful but gyroscope dependency needs solid fallback
5. **Sound Garden** — most complex (audio + visual), highest delight ceiling
