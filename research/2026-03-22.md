# Closr Block Research — March 22, 2026
## Focus: Color, Light & Ambient Mood — Gradients, Glow, and Emotional Atmospherics

Previous research covered: Typography (Mar 8), Particles (Mar 12), Scroll storytelling (Mar 13), Sound/audio (Mar 14), Device sensors (Mar 17), Photo effects (Mar 18), Gamification (Mar 20), 3D transforms (Mar 21). This session explores **color as emotion** — how gradients, ambient glow, light leaks, and shifting palettes can set mood in a scene without relying on text or imagery alone.

---

## 1. Breathing Gradient Backgrounds

**Concept:** A full-screen gradient that slowly shifts hue over time, like a living sky. Colors breathe — expanding warm, contracting cool — synced to heartbeat rhythm.

**Why it works for Closr:** Sets emotional tone before any content appears. A slow dawn-pink → deep-violet shift immediately says "intimate." Serves as ambient backdrop for any text/photo block.

**Technical approach:**
- CSS `@keyframes` animating `background-position` on large `background-size` gradient (400% 400%)
- Or `hsl()` via CSS custom properties for finer control
- GPU-composited, very lightweight
- Could tie to time-of-day (sunrise palette if morning, deep blues at night)

**Block idea: "Mood Wash"** — Background color evolves through a curated emotional palette while a single line of text fades in. The color IS the message.

---

## 2. Radial Glow / Spotlight Effect

**Concept:** Soft radial glow follows touch position or pulses from center — like holding a warm light source. Everything outside the glow is dim/desaturated.

**Why it works for Closr:** Creates intimacy — reading by candlelight. Touch literally brings warmth. Content within the glow feels secret, personal.

**Technical approach:**
- `radial-gradient` overlay, position updated via touch events
- Or CSS `mask-image: radial-gradient(...)` on dark overlay
- Combine with `filter: saturate()` — outside glow is desaturated, inside is full color
- Canvas alternative for complex scenes

**Block idea: "Warm Glow"** — Dark scene. Touch emanates golden light. Text/photos emerge from darkness as if discovered by candlelight.

---

## 3. Light Leak / Lens Flare Overlays

**Concept:** Animated light leaks (warm, dreamy color bleeds from film photography) drifting across the scene. Nostalgia, warmth, golden-hour romance.

**Why it works for Closr:** Instant emotional filter. Makes any photo feel like a treasured memory. Imperfection of light leaks feels human and analog.

**Technical approach:**
- Semi-transparent gradient divs with `mix-blend-mode: screen` or `overlay`
- Animate position/opacity with CSS keyframes
- 2-3 overlapping leak elements in warm amber/rose/gold
- Lightweight: just colored divs with blend modes

**Block idea: "Golden Hour"** — Photo/text scene with drifting light leaks creating late-afternoon sun warmth. Optional film grain texture overlay (CSS noise pattern).

## 4. Color Temperature Transitions (Cool → Warm)

**Concept:** Scene starts in cool blue/silver (loneliness, distance) and gradually warms to amber/rose (connection, love). Emotional arc told entirely through color.

**Why it works for Closr:** Perfect for LDR couples. Cold→warm mirrors "I miss you → but you're with me." No words needed.

**Technical approach:**
- CSS `filter: hue-rotate()` and `saturate()` animated over time or on scroll
- Or transition between gradient definitions using CSS custom properties
- Each tap could warm the scene slightly
- Combine with photo transforming from desaturated blue-tint to warm golden

**Block idea: "Closing the Distance"** — Opens cold and blue. Each tap (or auto over 5-8s) shifts warmer. Text appears at peak warmth. The color journey IS the emotion.

## 5. Neon Glow Text

**Concept:** Text with soft neon-sign glow — color bleeds outward, slight flicker, like a sign in a rainy city.

**Why it works for Closr:** Dramatic and atmospheric. "I love you" in flickering neon pink hits different. Cinematic mood.

**Technical approach:**
- CSS `text-shadow` with multiple layered shadows in decreasing opacity
- Flicker via `@keyframes` alternating opacity/shadow-blur
- Colors: neon pink (#ff6b9d), electric blue (#00d4ff), warm amber (#ffaa00)
- Optional `backdrop-filter: blur()` behind text for depth

**Block idea: "Neon Confession"** — Dark scene, rain-on-glass effect. Neon sign flickers to life, spelling message letter by letter. Glow reflects on rain drops. Could combine with existing rain-on-glass block.

## 6. Firefly / Floating Light Particles

**Concept:** Tiny warm dots drifting lazily — fireflies on a summer evening. Deliberately sparse and organic, not a full particle system.

**Why it works for Closr:** Magic without spectacle. Fireflies = summer nights, warmth, fleeting beauty. Sparseness feels intimate.

**Technical approach:**
- 15-30 small divs with `border-radius: 50%`, warm color, `box-shadow` for glow
- CSS keyframes with randomized durations/delays for organic drift
- Opacity pulse (brighten/dim) to simulate blinking
- Pure CSS, extremely lightweight — no canvas needed

**Block idea: "Summer Night"** — Dark scene with lazy fireflies. Message fades in as if written in warm air. Fireflies occasionally cluster near text.

---

## 7. Chromatic Aberration / RGB Split

**Concept:** Text/images with slight RGB channel offset that shifts on interaction — dreamy, surreal look.

**Why it works for Closr:** "Between-worlds" feeling for dream sequences, imagining a future together.

**Technical approach:**
- Three overlapping copies with one RGB channel each via `mix-blend-mode`
- Offset positions slightly, animate on touch
- Subtle: 2-4px offset max
- Resolves (channels merge) on tap — dream coming into focus

**Block idea: "Dream State"** — Gentle chromatic aberration resolves on tap — bringing a dream into focus. Pair with dreamy message about future plans.

## 8. Mood Ring / Emotion Color Mapping (System-Level)

**Concept:** Scene color scheme derives from emotional tone of message. AI maps sentiment to palette: love = deep rose, excitement = electric gold, nostalgia = warm sepia, longing = twilight blue.

**Why it works for Closr:** AI already generates content — extending to color makes every surprise feel uniquely calibrated. Colors become emotional data visualization.

**Technical approach:**
- AI tags message with emotion category during generation
- Map to curated CSS custom property palettes
- Apply to gradients, text, glow, particles
- Interpolate between palettes for mixed emotions
- Just CSS variables set by generation pipeline

**Not a standalone block but a SYSTEM** — Every block could use emotion-mapped palettes for visual-emotional coherence.

## Color Psychology Quick Reference for Closr:
- **Deep rose / blush pink** → romantic love, tenderness
- **Warm amber / gold** → nostalgia, warmth, comfort
- **Twilight blue / navy** → longing, depth, calm
- **Soft lavender** → dreaminess, gentleness
- **Electric pink / magenta** → passion, excitement
- **Deep violet** → mystery, intensity, devotion

## Performance Notes:
- CSS gradients and blend modes are GPU-composited — cheap
- Animating `filter` can be expensive; prefer `opacity` and `transform`
- `mix-blend-mode` has excellent support now
- CSS custom properties make palette-swapping trivial at runtime

## Integration with Existing Blocks:
- Existing blocks (rain-on-glass, snow-globe, aurora-dreams) would benefit from **dynamic color palettes** rather than hardcoded colors
- "Mood Wash" could be a universal backdrop layer for ALL blocks
- Light leak overlays could be a composable layer for any photo-based block
- The emotion-mapping system could unify the entire visual language of Closr surprises
