# Closr Block Research — March 8, 2026
## Focus: Typography & Text Animation Treatments

Previous research: none found. Existing block: `memory-polaroid.html` (photo-centric, Polaroid stack with swipe). This session explores **text-first** visual treatments — how words themselves become the spectacle.

---

### 1. Handwriting Reveal (SVG stroke-dashoffset)

**Technique:** SVG paths traced with `stroke-dasharray` + `stroke-dashoffset` animation, making text appear to be hand-drawn in real-time.

**Why it works for Closr:** A Love Letter block that literally writes itself in front of the viewer's eyes. The intimacy of watching handwriting appear — as if the sender is writing it right now — creates an emotional moment no static text can match.

**Implementation notes:**
- Convert a script font (e.g., Dancing Script, Pacifico) to SVG paths
- Animate `stroke-dashoffset` from full length to 0
- Add a slight pen-trail glow (soft drop-shadow on the stroke tip)
- Speed: ~2-3 seconds per line feels natural, not rushed
- Pair with soft paper texture background (`#f5f0e8`, subtle grain)

**Best for block types:** Love Letter, Appreciation Report, Comfort

---

### 2. Word-by-Word Fade-In with Emphasis Pulse

**Technique:** Text appears one word at a time with a soft fade + slight Y-translate. Key emotional words get a brief scale pulse (1.0 → 1.08 → 1.0) and a warm color shift (white → rose gold → white).

**Why it works for Closr:** Forces the reader to experience the message at the sender's intended pace. No skimming. Every word lands. The emphasis pulse on emotional words ("love," "always," "us," their partner's name) creates micro-moments of intensity.

**Implementation notes:**
- Stagger: 120-180ms per word
- Emphasis detection: AI marks 2-3 key words in the content slot with `<em>` tags
- Emphasis animation: `color: #e8a4b8` pulse + `text-shadow: 0 0 20px rgba(232,164,184,0.4)`
- Background: deep navy `#0a0e1a` for contrast
- Font: serif for emotional weight (Playfair Display or Cormorant Garamond)

**Best for block types:** Hype Up, Comfort, Love Letter

### 3. Typewriter with Cursor + Backspace "Corrections"

**Technique:** Classic typewriter effect, but the AI-written text includes deliberate "mistakes" that get backspaced and corrected — as if the sender is searching for the perfect words.

**Example flow:**
```
"You are incredi... [backspace] ...the most amaz... [backspace] ...there aren't words big enough for what you are to me."
```

**Why it works for Closr:** The vulnerability of watching someone struggle to express their feelings is more powerful than polished text. It humanizes the AI-generated content. The "mistakes" are scripted by Claude as part of the content slot.

**Implementation notes:**
- Monospace or slightly imperfect typewriter font (Special Elite, Courier Prime)
- Typing speed: 60-80ms per character, backspace slightly faster (40ms)
- Blinking cursor: `|` with 530ms blink interval
- Subtle key-click audio (optional, CSS-only version works without)
- Dark background with slight CRT scan-line overlay for atmosphere

**Best for block types:** Love Letter, Inside Joke, Appreciation Report

### 4. Kinetic Typography — Words as Architecture

**Technique:** Words arranged spatially, not linearly. Text animates into position from different directions, building a visual composition. Think: movie title sequences.

**Example:** The word "US" huge and centered, with smaller phrases orbiting in:
- "late night calls" drifts in from left
- "inside jokes nobody gets" slides up from bottom
- "3,847 miles" fades in above
- All settle into a constellation around "US"

**Why it works for Closr:** Transforms a list of relationship details into a visual poem. Each phrase is a data point from the couple's answers, but arranged artistically it becomes art, not a database query.

**Implementation notes:**
- CSS transforms + absolute positioning
- Each phrase gets unique: entry direction, font size, opacity, rotation (-3° to 3°)
- Central word: bold, large (clamp(2rem, 8vw, 4rem))
- Satellite phrases: lighter weight, varied sizes
- Final state holds for 3s before gentle parallax drift begins
- Color palette: cream text on charcoal, one accent color (rose/gold)

**Best for block types:** Memory Replay, Story, Appreciation Report, Future Vision

### 5. Letter-Spacing Reveal (The "Breathing" Text)

**Technique:** Text starts with extreme letter-spacing (fully spread across screen, nearly illegible) and slowly contracts to readable form — like the message is condensing out of thin air, or taking a breath before speaking.

**Why it works for Closr:** Creates a moment of anticipation. The viewer can almost-but-not-quite read it, leaning in mentally. When it resolves, the message hits harder because they've been waiting for it.

**Implementation notes:**
- Start: `letter-spacing: 2em; opacity: 0.3`
- End: `letter-spacing: normal; opacity: 1`
- Duration: 2.5-3s with `cubic-bezier(0.25, 0.1, 0.25, 1)` easing
- Works best with short, punchy lines (1-6 words)
- Stack multiple lines with staggered timing for longer messages
- Subtle blur that clears as spacing normalizes: `filter: blur(2px)` → `blur(0)`

**Best for block types:** Countdown, Celebration, Hype Up

---

### 6. Redacted/Classified Reveal

**Technique:** Text appears fully redacted (black bars over words on a document-style background). Tap/swipe to "declassify" — bars slide away one by one, revealing the message beneath.

**Why it works for Closr:** Gamifies the reading experience. Each reveal is a tiny dopamine hit. Works brilliantly for Inside Joke blocks ("CLASSIFIED: Things Only We Know") or playful Appreciation Reports styled as dossiers.

**Implementation notes:**
- Background: off-white with faint grid lines, "CLASSIFIED" watermark at 15% opacity
- Redaction bars: `background: #1a1a1a` with slight rounded corners
- Reveal: bars slide right with slight bounce, text underneath has highlight flash
- Optional: "TOP SECRET" stamp in corner, rotated 12°
- Font: monospace for the document feel (IBM Plex Mono)
- Progressive reveal on scroll or tap

**Best for block types:** Inside Joke, Appreciation Report, Story

---

### 7. Gradient Text Flow (Living Color)

**Technique:** Text rendered with animated gradient that flows continuously through the letters, like liquid color moving through the words. Not a static gradient — a living, breathing color shift.

**Why it works for Closr:** Mesmerizing without being distracting. Makes even simple text feel magical and alive. Works as a treatment layer on top of other techniques.

**Implementation notes:**
- `background: linear-gradient(90deg, #e8a4b8, #a4c4e8, #e8d4a4, #e8a4b8)`
- `background-size: 300% 100%`
- `background-clip: text; -webkit-background-clip: text; color: transparent`
- `animation: gradientFlow 6s ease infinite`
- Keyframes shift `background-position` from `0% 50%` → `100% 50%` → `0% 50%`
- Pair with large display font for maximum effect

**Best for block types:** Celebration, Future Vision, Hype Up
